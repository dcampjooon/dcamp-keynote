## 키노트 보기

`2026-05-31/index.html`을 브라우저에서 엽니다.

- `←` / `→` (또는 클릭·`Space`) : 슬라이드 이동
- `F` : 전체화면
- 하단 점 클릭 : 특정 슬라이드로 점프
- URL `#번호` : 해당 슬라이드로 바로 이동 (예: `index.html#5`)

외부 의존성은 Pretendard 폰트 CDN 하나뿐이라, 더블클릭만으로 풀스크린 발표가 됩니다.
